# beanstalkd rpmbuild

## Create an RPM Build Environment

```
yum install rpmdevtools
rpmdev-setuptree
```

## Install Prerequisites for RPM Creation

```
yum groupinstall 'Development Tools'
```

## Build the RPM

```
./build.sh
```
